# SEW_Projekt_3AHIT_Karlhuber
Das Projekt ist ein Verwaltungsystem für eine Stadt. Man kann sehen wie viele Personen in der Stadt sind und wie viele rein dürfen, wann die Müllabfuhr fährt und wo, ob Veranstaltungen sind und noch mehr Sachen um eine Stadt zu kontrollieren und diese Daten sollte man dann speichern können.
